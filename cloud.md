# KROVA Scaffold Notes

This repo is intentionally reduced to the reusable scaffold:

- Next.js App Router UI
- Postgres with Drizzle
- Better Auth magic-link authentication
- pg-boss background worker
- Email outbox with SMTP delivery (nodemailer)
- Orbit admin with user, queue, and email views

Keep product-specific features out of the scaffold unless they are examples for these primitives.
